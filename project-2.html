NBA Player Performance Analytics

Using regression models and PCA to understand what actually drives NBA player performance — and which statistics carry the most signal.

Python NumPy PCA Regression Analysis Data Visualization

Overview

NBA player statistics are notoriously multicollinear — points, assists, rebounds, and shooting percentages all correlate with each other in complex ways. This project aimed to cut through that noise using dimensionality reduction and regression to identify which underlying performance factors are truly independent predictors of player value.

The Data

I worked with a dataset of per-game player statistics across multiple NBA seasons, including traditional box score stats (points, rebounds, assists, steals, blocks, turnovers) and advanced metrics (PER, true shooting %, usage rate).

Key data challenges:

  • Players who were traded mid-season appeared multiple times with split stats — required aggregation logic
  • Rookie seasons and injury-shortened seasons created outliers with low sample sizes that needed filtering
  • Missing values in advanced metrics for older seasons required decisions about imputation vs. exclusion

My Approach

I first built linear regression models to establish baselines for predicting player efficiency rating (PER) from raw stats. Then I applied nonlinear regression (polynomial features) to capture interaction effects between statistics.

The more interesting analysis was PCA via SVD using NumPy. By decomposing the player statistics matrix, I identified the principal components — the underlying dimensions of variation that explain most of the differences between players.

The first principal component separated high-usage scorers from low-usage role players. The second separated perimeter players (high assists, steals) from frontcourt players (high rebounds, blocks). Together, two components explained over 70% of variance.

Results

Linear regression on raw stats achieved an R² of 0.74 for predicting PER. Adding polynomial terms improved this to 0.81, with the biggest gains from interaction terms between usage rate and shooting efficiency.

The PCA visualization revealed intuitive player clusters: traditional big men, point guards, 3-and-D wings, and high-usage scorers occupied distinct regions of the component space — consistent with how basketball analysts categorize player archetypes.
